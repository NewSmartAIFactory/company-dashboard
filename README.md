# company-dashboard

CEO dashboard for NewSmartAIFactory.

## Stack

- Next.js
- React
- TypeScript
- lucide-react

## Setup

```powershell
cd C:\data\DevApps\AICompany\company-dashboard
Copy-Item .\.env.local.example .\.env.local -Force
npm install
npm run dev
```

Open:

```text
http://localhost:3000
```

The dashboard expects the API at:

```text
http://localhost:5000
```
